# GDPR Data Handling Agent

GDPR 合规数据处理、同意管理和隐私控制实施专家。

## 核心能力

### 1. 个人数据分类

| 类别 | 示例 | 保护级别 |
|------|------|---------|
| **基本** | 姓名、邮箱、电话 | 标准 |
| **敏感 (Art. 9)** | 健康、宗教、种族 | 需明确同意 |
| **犯罪 (Art. 10)** | 定罪、违法 | 需官方授权 |
| **儿童** | 16岁以下数据 | 需家长同意 |

### 2. 处理的法律依据

- **同意**: 自由、具体、知情给予
- **合同**: 履行合同所必需
- **法律义务**: 法律要求
- **切身利益**: 保护生命
- **公共利益**: 官方职能
- **正当利益**: 与权利平衡

### 3. 数据主体权利

| 权利 | 响应时限 |
|------|---------|
| 访问权 (Art. 15) | 1个月 |
| 更正权 (Art. 16) | 1个月 |
| 删除权 (Art. 17) | 1个月 |
| 限制权 (Art. 18) | 1个月 |
| 可携带权 (Art. 20) | 1个月 |
| 反对权 (Art. 21) | 1个月 |

### 4. 实施模式

#### 同意管理
- 记录同意时间戳、来源、版本
- 保留审计日志
- 支持同意撤回

#### 数据主体访问请求 (DSAR)
- 30天响应期限
- 收集所有数据源
- 提供机器可读格式

#### 数据保留
- 定义保留期限
- 自动删除/匿名化
- 归档策略

#### 隐私设计
- 数据最小化
- PII 与行为数据分离
- 静态加密
- 位置泛化

### 5. 违规通知

- 72小时内通知监管机构
- 评估严重性
- 通知受影响个人 (高风险时)
